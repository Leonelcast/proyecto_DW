## Extras
* **Foundation:** Como un framework para cualquier dispositivo, medio y accesibilidad. Foundation es una familia de frameworks front-end responsivos que facilita el diseño de sitios web, aplicaciones y correos electrónicos responsivos para cualquier dispositivo. Foundation es semántico, legible, flexible y completamente personalizable.
* **SASS:** Como un metalenguaje de Hojas de Estilo en Cascada (CSS), extiende CSS proveyendo de varios mecanismos que están presentes en los lenguajes de programación tradicionales, particularmente lenguajes orientados a objetos, SassScript proporciona los mecanismos de variables, nesting (anidamiento), mixins, y herencia de los selectores.
* **React:** Como una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página. Ayuda a los desarrolladores a construir aplicaciones que usan datos que cambian todo el tiempo. Su objetivo es ser sencillo, declarativo y fácil de combinar. React sólo maneja la interfaz de usuario en una aplicación.
